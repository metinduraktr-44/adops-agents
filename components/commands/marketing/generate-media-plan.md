# /generate-media-plan
Input: objective, budget, market, timeframe, constraints ($ARGUMENTS).
Output a media plan: channel split (%/amount), rationale, KPIs per channel,
flighting, and measurement plan. Detect market (e.g. TR) and adjust channel mix.
Keep it table-first, copy-paste ready. State assumptions.
